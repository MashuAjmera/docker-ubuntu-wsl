# Start Docker Service in Ubuntu (WSL 2) 

## Open WSL in Windows file explorer
In the address bar type: `\\wsl$\`
Go to your Distribution

## Run Script
Copy the script (`docker.sh`) inside your Ubuntu (WSL).  
Run using `sh docker.sh`
